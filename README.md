# Sazandora

■概要  
Unreal Engineを用いたマルチプレイゲームです。プレイヤー同士で遊ぶことを想定した作品です。

■使用技術  
・Unreal Engine  
・C++

■担当  
・プレイヤー挙動の実装  
・マルチプレイにおける同期処理  
・NPCごとに異なるアイテム取得処理

■工夫した点  
サーバーとクライアントの役割分担を見直し、プレイヤーの状態が正しく同期されない問題を解決しました。  
ログ出力やデバッグを繰り返し行い、処理の順序や取得タイミングを調整することで安定した動作を実現しました。

■補足  
NPCに一定距離まで近づき、視線を合わせて入力することでアイテムを取得できる仕組みを実装しています。
