# Sazandora

## ■ 概要
Unreal Engineを用いたマルチプレイゲームです。  
複数プレイヤーでのプレイを想定し、同期処理やプレイヤー体験に関わる機能の実装を行いました。

## ■ 使用技術
- Unreal Engine
- C++

## ■ 担当
- プレイヤー挙動の実装
- マルチプレイにおける同期処理
- NPCごとに異なるアイテム取得処理の実装

## ■ 工夫した点
サーバーとクライアント間でプレイヤーの状態が正しく同期されない問題に対し、  
処理の実行主体（サーバー／クライアント）の見直しを行いました。  

ログ出力やデバッグを繰り返しながら原因を特定し、  
処理の順序やデータ取得タイミングを調整することで、安定した同期を実現しました。

## ■ 実装内容
NPCに一定距離まで近づき、視線を合わせた上で入力することで  
アイテムを取得できるインタラクションを実装しています。

## ■ ポイント
- マルチプレイにおける同期処理の理解
- サーバー／クライアント間の役割分担の設計
- 問題の原因特定から解決までの対応力
